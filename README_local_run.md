# Nisha Rajaput | Local Run Guide

## Start the app
Double-click run_local_app.bat

Or run from PowerShell:
```powershell
./run_local_app.bat
```

Then open:
```text
http://localhost:8501
```

## Stop the app
Double-click stop_local_app.bat

Or run from PowerShell:
```powershell
./stop_local_app.bat
```
